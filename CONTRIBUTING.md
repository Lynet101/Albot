## How Can I Contribute?

### Reporting bugs
For reporting bugs, we prefer githubs inbuilt system. Please provide a clear description of the issues, along with the steps to reproduce

### Contributing code
If you want to contribute code, please create a fork off of the main repository, and run extensive tests on a testing discord server. Once you know your function is working, feel free to create a pull request. Your code will now be reviewed and/or refactored by @lynet_101 before either merged or discarded
